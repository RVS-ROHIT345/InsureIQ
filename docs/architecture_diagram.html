Why it's trustworthy

Numbers vs. prose. Every figure a user sees — return rate, maturity value, benchmarks — is computed by deterministic Python, never guessed by the model. Gemini writes only the narrative around numbers it can't touch. That's what makes the verdict dependable, and testable by a 192-test suite that needs no API key.
